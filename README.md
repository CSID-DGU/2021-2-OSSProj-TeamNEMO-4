# How To play

```
 > pip install -r requirements.txt
 > python main.py
```

<hr>

# Features

> - 4 가지 미니게임 플레이 가능 [무궁화, 달고나, 줄다리기, 구슬 홀짝]
> - 3 가지 모드 [ 무한 모드, 최고 기록 모드, 선택 모드 ]
> - 상위 5위 내 기록시 외부 DB 에 스코어 기록
> - 랭킹 보기 화면에서 모드 별 상위 5위 까지의 스코어 확인 가능
